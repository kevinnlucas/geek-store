# 🛒 Geek Store - Backend & Security Project

Este projeto acadêmico foi desenvolvido para demonstrar práticas de **Segurança da Informação** e **Arquitetura de Sistemas** no desenvolvimento web. O foco principal é a proteção de dados sensíveis e a estruturação de uma API fictícia para consumo no frontend.

## 🛡️ Pilares de Segurança Aplicados

* **Gestão de Segredos:** Utilização de arquivos `.env.local` para armazenar chaves de API e URLs de banco de dados (PostgreSQL/Supabase).
* **Blindagem de Repositório:** Configuração rigorosa de `.gitignore` para impedir o vazamento de credenciais sensíveis no GitHub.
* **Integridade de Dados:** Estruturação de dados via `produtos.json` seguindo padrões de modelagem de banco de dados acadêmicos.

## 🚀 Tecnologias e Ferramentas

* **Frontend:** HTML5, CSS3 (Modern Dark Theme) e JavaScript Assíncrono (Fetch API).
* **Backend Simulado:** Banco de dados baseado em objetos JSON.
* **Versionamento:** Git & GitHub com fluxo de commits organizados.
* **Infraestrutura:** Planejado para execução em ambientes otimizados (Dell Inspiron 15 3567) e bancos de dados SQL[cite: 2].

## 📋 Como Executar o Projeto

1. Clone o repositório.
2. Certifique-se de que o arquivo `.env.local` esteja configurado em sua máquina local (conforme os padrões de segurança estabelecidos).
3. Abra o `index.html` utilizando um servidor local (como a extensão Live Server do VS Code).
